A chart about movies (and buttons)

An example of the use of buttons for basically interacting with SVG elements.

Bootstrap

We start from the simplest Bootstrap template –seen in the lesson about web programming– to which we add just buttons and our content (a chart about movies). Buttons are configured to add or remove the class "idle" to our elements so that when they have it, our CSS makes them appear as grey.

–

JavaScript

In this example, buttons work with the use of JavaScript. More specifically, we take advantages of jQuery, a commonly used library. With "library" here we mean a set of previously written functions ready to be used in our projects.

If you are successfully using Bootstrap, no further action is required to install jQuery: since it is necessary to run the framework, it has been already imported/linked while setting up the template (look at the end of your HTML file).

jQuery is composed by several functions, but here we just use the ones called "click( )" and "toggleClass( )".
This latter is able to check the class attribute of an element and add or remove a className. We can perform this action on the same clicked element and/or to any other element in the page.


In the JavaScript file other similar rules had been created, one for each movie category.

–

CSS

When an element is active it DOES NOT HAVE the "idle" class and it just keep the same rules of the generated SVG. When it does, then new CSS rules are applied and their appearences change.


Every movie is represented with a circle colour-coded depending on its category. Movies under the same category have been grouped and renamed in Illustrator: the names we gave are turned into HTML IDs once exported, so we can refer to those elements with: #drama, #adventure, #action, etc.
When a group has the class "idle", the CSS changes the colour fill of the circles into grey. When that class is removed, the new CSS rule disappear and the fill colour switches back to the original one, automatically.
